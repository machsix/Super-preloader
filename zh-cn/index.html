# 简介

自动翻页脚本，由ywzhaiqi/Super_preloaderPlus (opens new window)修改而来。在此首先对作者swdyh (opens new window), ywzhaiqi (opens new window)和NLF (opens new window)致以谢意。swdyh 仍在频繁的更新规则 (opens new window)以及开发扩展AutoPagerize (opens new window)。

相比AutoPagerize (opens new window)， 该脚本的主要优势在于规则的定义和 DOM 的处理更加灵活。你不仅可以使用xpath selector (opens new window)定义规则，还可以直接使用JavaScript (opens new window)提取元素、修改页面（比如去广告DocumentFilter、增加新功能）等。另一方面，鉴于 swdyh 维护的规则主要针对日文用户，我也希望该脚本及相关规则能惠及中文、英文用户。
