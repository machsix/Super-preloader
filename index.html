# Introduction

An userscript to preload next page and join pages together.

The script is forked from ywzhaiqi/Super_preloaderPlus (opens new window). Thanks to the original author swdyh (opens new window), ywzhaiqi (opens new window) and NLF (opens new window). swdyh is still actively improving rules (opens new window) and developing the Chrome extension, AutoPagerize (opens new window)

This userscript is more powerful than AutoPagerize (opens new window) in the sense that it has more flexibility for the method to define rules. You can not only use xpath selector (opens new window) link to derive the url for the next page, but also use JavaScript (opens new window) function to achieve the goal. You are also able to correct the content if the automatic rule is not perfect since we provide DocumentFilter.

The other reason for the existence of this userscript is because someone needs to maintain the rules for Chinese and English users since swdyh’s rules are mainly for Japanese websites. Don’t be afraid of the fact that most feedbacks are in Chinese. I would like to add rules for English users if I could. Feel free to leave feedback.
